# SMSimulator_API
Backend API for StockMarketSimulator

# How to set up and connect SMSimulator API with SMSimulator Website
1. First clone both repository
2. run **npm install** on both cloned repositry to install dependencies
3. run **npm start** on SMSimulator_API directory to start backend server
4. run **npm start** on SMSimulator_Website directory to start react server

This is temporary, we can later put a script to run it all in one go
